# linaix-os

![logo](./docs/img/logo/logo.png)

a unix-like os.

# try the linaix-os

## build

``` shell
xmake build
```

## run

``` shell
# You should build before running
xmake run
```

## development

``` shell
xmake project -k compile_commands
```
